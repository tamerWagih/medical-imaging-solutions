Key Features Extracted

  1. HRS1_Volume - Volume of highest HRS region
  2. HRS6_Volume - Volume of medium HRS region
  3. HRS7_6_Ratio - Ratio of HRS7/HRS6 volumes
  4. ADC_Prostate_Mean - Mean ADC in prostate
  5. ADC_HRS1_Mean - Mean ADC in HRS1 mask
  6. ADC_HRS6_Mean - Mean ADC in HRS6 mask

Experiment Parameters

  • WashOutUpper: 2.0 to 4.5 (affects DCE-based features)
  • ADC PZ Thresholds: High/Medium/Low (affects ADC-based features)
  • See Experiment_Plan.csv for all 36 configurations

What T-Tests Do

Compare feature values between two patient groups (label 0 vs label 1) to find features that show significant differences (p < 0.05).

âš ï¸ Troubleshooting

Problem: "No MAST folders found"

  • Check rootDir path is correct
  • Verify folders start with "MAST"

Problem: "Labels.csv not found"

  • Use absolute path
  • Check file exists and has mast_id,label header

Problem: "All p-values are NaN"

  • Ensure both groups (0 and 1) have patients
  • Check feature values aren't all NaN

Problem: Identical values across experiments

  • This is normal! Some patients naturally have similar values
  • Identical parameters = identical results (algorithm is deterministic)
