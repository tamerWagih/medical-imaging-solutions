Root Cause Analysis

Debugging Process:

  1. Reviewed HRS Java code implementation
  2. Traced DCEScore calculation
  3. Analyzed patient MAST007 as example

Key Finding:

DCEScore calculation explains similarities

DCEScore = (Math.max(Math.min(DCEFeatureSum, upper), lower) - lower) / (upper - lower)

When DCEFeatureSum values are similar, DCEScore becomes identical, leading to identical HRS features.

Technical Implementation

HRS Algorithm Flow:

1. DCE Processing:

  • Calculate DCEFeatureSum from DCE data
  • Apply WashOutUpper threshold
  • Compute DCEScore = (max(min(DCEFeatureSum, upper), lower) - lower) / (upper - lower)

2. HRS Calculation:

  • wNorm = min(wWP.get(i, 0) / wNthPercentile, 1)
  • hrsDCE.set(i, 0, wNorm * DCEScore)

3. Feature Extraction:

  • HRS volumes calculated from final HRS masks
  • ADC means calculated within HRS regions
  • Ratios computed from volume relationships
