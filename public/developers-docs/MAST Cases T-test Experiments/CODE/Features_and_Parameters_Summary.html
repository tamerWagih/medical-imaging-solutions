Why Values Are Identical

Root Cause: Experiments with identical parameter combinations produce identical results.

Example Identical Groups:

  • Group 1: Experiments 1,5,6,20,21,22,23,24 (WashOutUpper=4, PZ=850/1050/1200)
  • Group 2: Experiments 2,7,8,25,28,31,34 (WashOutUpper=2.5, various PZ values)
  • Group 3: Experiments 3,9,10,26,29,32,35 (WashOutUpper=3, various PZ values)
  • Group 4: Experiments 4,11,12,27,30,33,36 (WashOutUpper=3.5, various PZ values)

Solution

Check that each experiment uses different parameter combinations - the algorithm is deterministic, so identical parameters = identical results.
